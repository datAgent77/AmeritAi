# Vion AI - Kurumsal Bilgi Dokümanı

## Vion AI Nedir?

Vion AI, işletmelerin müşteri deneyimini iyileştirmek için tasarlanmış yapay zeka destekli bir chatbot platformudur. Web sitelerine kolayca entegre edilebilen akıllı asistanlar oluşturmanızı sağlar. Hem Türkçe hem İngilizce tam destek sunar.

**Resmi Web Sitesi:** https://getvion.com

## Nasıl Başlanır?

### 1. Kayıt Olun
https://getvion.com/signup adresinden ücretsiz hesap oluşturun.

### 2. Sektörünüzü Seçin
İşletmenize uygun sektörü seçin, AI otomatik olarak optimize edilecektir.

### 3. Chatbot'u Eğitin
Web sitenizin URL'sini girin, Vion AI içeriğinizi öğrenecek.

### 4. Widget'ı Kurun
Size verilen embed kodunu web sitenize ekleyin.

```html
<script src="https://getvion.com/widget.js" data-chatbot-id="YOUR_ID"></script>
```

### 5. Canlıya Alın
Chatbot'unuz anında çalışmaya başlar!

### Kaç dil destekleniyor?
Şu an Türkçe ve İngilizce tam destek mevcuttur. Chatbot otomatik dil algılama yapabilir.

Evet. Tüm veriler şifreli olarak saklanır. KVKK ve GDPR uyumludur.

Evet, 14 gün ücretsiz deneme sunuyoruz. Kredi kartı gerekmez.

### Mevcut chatbot'umu Vion'a taşıyabilir miyim?
Evet, mevcut konuşma geçmişinizi ve eğitim verilerinizi taşımanıza yardımcı oluruz.

### API erişimi var mı?
Pro ve Enterprise planlarda tam API erişimi mevcuttur.

### Mobil uyumlu mu?
Evet, widget responsive tasarıma sahiptir ve mobil cihazlarda mükemmel çalışır.
